# Zilun Zhang Academic Resume

Static GitHub Pages site for `zilunzhang.github.io`.

## Local Preview

Open `index.html` directly in a browser, or run a static server from this folder.

## Publish

Create a GitHub repository named `zilunzhang.github.io`, copy these files to the repository root, then push to the default branch. GitHub Pages will serve it at `https://zilunzhang.github.io`.

## Notes

- The public page uses email, GitHub, and Google Scholar links.
- Phone number is intentionally omitted for public publishing.
- The avatar is loaded from the public GitHub profile image URL.
